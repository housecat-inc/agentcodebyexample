## Rationale

The modern computer terminal evolved from electromechanical teletypes (1940s-50s) and video-based CRT displays (1960s-70s). Inputting commands with a keyboard and outputting feedback as text is a universal capability for every computer system. 

LLMs are excellent at reading text and inferring what text should come next. This has lead every major agentic coding provider to offer a terminal UI (TUI) and command line interface (CLI) for their agents to control a computer through its terminal "shell" and to read and write code.

By comparison, browser use and computer use are magnitudes harder to model and provider inference, and graphical user interfaces (GUI) to manage a computer are not universally available and are limited in their capabilities.

"Hello world" was first published in *The C Programming Language* in 1978 by Brian Kernighan. Kernighan helped develop the Go programming language starting in 2012 and published *The Go Programming Language* in 2015.
