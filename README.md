# Agent Code by Example

Content and tooling for Agent Code by Example, a reference that teaches how to code with agents via annotated example programs.

## Rationale

Fundamental philosophies of software development are well understood through 50+ years of computer science and engineering.

But AI and agentic coding changed everything.

There is a new wave of people writing software for the first time. The new tools and techniques are novel and rapidly evolving that even seasoned developers have a lot to re-learn and un-learn.

*Agent Code by Example* is a hands-on introduction to coding with annotated example programs.
